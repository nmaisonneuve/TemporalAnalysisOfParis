• Description Discovered square visual elements of facades of buildings in Paris that are discriminative (unique) to a time period and representative (present in lots of building) of this period. The visual elements are ranked according to 2 attributes: Purity and Frequency and grouped (or not) according to the size of the visual element (multi-scale discovery method). Data: Google Street view images (no ground floor) + cadastre data from Paris. Colored border of each visual element = time period.
  • List of time periods / experiments
    - 1800 1801-1850 1851-1914 1915-1939 1940-1967 1968-1975 1976-1981 1982-1989 1990-1999 2000 +
  • Visual elements ranked by: (default = harmonic mean)
    - Purity (% of detections from positive images, ~ precision),
    - Frequency (% of positive images with at least one detection of this visual element, ~ recall) ,
    - Harmonic mean of both
  • Group by: (default = grouped)
    - Patch size
    - no group
  • Associative/Hebbian learning & co-occurence graph: To discover associations between visual elements i.e. their co-appearance inside a shared context, we build co-occurrance graph with 2 definitions of context
    • context = shared image.
    • context = shared region (overlap)
