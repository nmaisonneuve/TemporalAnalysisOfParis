• Description: The top most discriminative clusters of visual elements coming from the facade of buildings in Paris
  • Label / discriminative attribute: time period of building construction
  • Positive label / investigated time period:
  • Colored border of each visual element
    • Green = visual element coming from a building of the investigated time period (positive label)
    • Red = visual element coming from buildings of other time periods (negative label)
  • Clusters ordered by purity score (purity ratio = nb of visual elements having a positive label / number of elements in the cluster).
  • to change the experiment, change {exp_name} in the url i.e.g index.html?experiment={exp_name}
